# CMM.dev

Terminal-style engineering portfolio built with [gridland](https://github.com/thoughtfulllc/gridland) — a TUI framework that renders React components on HTML5 Canvas.

## Stack

- Vite 6 + React 19 + TypeScript
- @gridland/web (canvas TUI renderer + vite plugin)
- @gridland/utils (keyboard hooks)

## Development

```bash
bun install
bun run dev
```

## Keyboard Navigation

| Key | Screen |
|-----|--------|
| `h` / `Esc` | Home |
| `p` | Projects |
| `a` | About |
